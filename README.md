# 🚀 Mahananta | Personal Portfolio

![Portfolio Preview](./react-portfolio/public/apple-touch-icon.png)

A modern, highly interactive, and fully responsive personal portfolio website built to showcase projects, skills, and professional experience.

## ✨ Features

- **🎨 Modern UI/UX:** Clean, elegant, and aesthetic design with premium glassmorphism effects.
- **🌓 Dark Mode Ready:** Seamlessly switches between Light and Dark themes, automatically respecting your system preferences (or saving your manual choice).
- **⚡ Blazing Fast:** Powered by Vite + React for lightning-fast load times and optimized performance.
- **🪄 Smooth Animations:** Advanced scroll-triggered animations and micro-interactions powered by GSAP.
- **📱 100% Responsive:** Flawless experience across all devices (Mobile, Tablet, Desktop).
- **🌊 Smooth Scrolling:** Integrated with Lenis for a buttery-smooth scrolling experience.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v3](https://tailwindcss.com/)
- **Animations:** [GSAP (GreenSock)](https://gsap.com/)
- **Smooth Scroll:** [@studio-freight/lenis](https://github.com/studio-freight/lenis)
- **Icons:** Custom SVG Icons & Lucide

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ananta1705/mahananta.dev.git
```

2. Navigate into the project directory:
```bash
cd mahananta.dev/react-portfolio
```

3. Install the dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open [http://localhost:5173](http://localhost:5173) in your browser to view the website.

## 🌐 Deployment

This project is optimized and ready to be deployed on platforms like **Vercel** or **Netlify**. 

**Important:** When importing this repository to Vercel/Netlify, make sure to set the **Root Directory** to `react-portfolio`.
